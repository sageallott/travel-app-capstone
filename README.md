# Travel App Capstone

## Overview
This project is an asynchronous web app that uses multiple web APIs to pull location information. 

## Instructions
This web application pulls user inputted location data.  The location is found using the geoNames API to find the postal code, and then calls the weatherbit API and the Pixabay API to add weather information and a photo of the location. The days until the trip is displayed, as well has how long the trip will be, with an added end date.
